# Tracquet

Tracquet is a Flask Progressive Web App (PWA) that provides tennis fans with a centralised space to view live tennis data including match scores, player rankings, and individual player information for both men's and women's tennis.

## Status

- **Current Sprint:** Sprint 2 — API Connection
- **Last Increment:** Sprint 1 complete — working login, dashboard, navbar, and placeholder pages delivered.
- **Next Planned Increment:** Sprint 3 — API integration into Live Matches, Rankings, and Player Profile pages.

## How to Run

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python main.py`
4. Navigate to `http://localhost:5000` in your browser
