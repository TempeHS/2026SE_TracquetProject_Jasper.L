# Tracquet

Tracquet is a Flask Progressive Web App (PWA) that provides tennis fans with a centralised space to view live tennis data including match scores, player rankings, and individual player information for both men's and women's tennis.

## Status

- **Current Sprint:** Sprint 3 — API Integration
- **Last Increment:** Sprint 3 complete — API data integrated into Live Matches, Rankings (ATP + WTA), and Player Profile pages. Dashboard previews, player search, and response caching delivered.
- **Next Planned Increment:** Sprint 4 — Testing: client feedback, UAT, and PWA testing/debugging.

## How to Run

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your API key: `API_TENNIS_KEY=your_key_here`
4. Run the app: `python main.py`
5. Navigate to `http://localhost:5000` in your browser
